# Members Only

## Features:

1. Login and Sign Up and Logout
2. Create messages
3. Admin users can delete and update messages
4. Normal users can look at chats with author and timestamp
5. Guest users can look at chats without author and timestamps
6. Membership users can look at messages with other members only, like a special cool kids club
7. Admin users can look at both normal users and membership users messages

## Steps:

1. Run `npm i`
2. Install postgresql, and make a database
3. Update the .env.template file with information, and rename it to .env
4. Run `npm run db`
5. Run `npm run dev`
6. Go to http://localhost:3000
7. Enjoy!
